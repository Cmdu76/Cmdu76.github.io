Hi ! Welcome to my first serie of tutorials.

It address to people with basic programming knowledge who want to start learning how games are made.

I often don't like "tutorials" but I tried to help with important-to-learn things in a "correct" order so that the learning curve should be nice.

The language we will use here is C++ with the famous SFML Library.
If you don't know C++ but come from another OOP language it should be good enough.
If you are new to programming, you should find a correct OOP language start before.

I split this tutorials into three major parts and each is subdivided into smaller part.
